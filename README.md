## 功能特性
* [x] 使用 JavaScript 开发，可在浏览器中运行
* [x] 支持编辑器的常用功能（编辑、多点编辑、折叠、历史记录、代码高亮、语法提示等）
* [x] 支持代码的常见语法错误检查
* [x] 支持百万行级别的代码
* [x] 可扩展其他语言
